# Brewline

Worldwide online coffee ordering app. Single-origin beans from 12 countries, shipped to 47.

Built with Next.js 16 (App Router), React 19, and Tailwind 4.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Features

- 12 single-origin coffees from 5 continents
- Region filter + search by name, origin, or tasting notes
- Cart persisted in `localStorage`
- Worldwide checkout with region-based shipping (free over $40)
- Order confirmation with generated order ID
